# Изображения для лендинга ShuKnow

Поместите сюда скриншоты продукта для лендинга.

## Необходимые изображения

### hero.webp
- **Размер:** ~1200×700px
- **Описание:** Скриншот основного интерфейса (чат + sidebar с папками)
- **Где используется:** Hero-секция лендинга

### feature-sort.webp
- **Размер:** ~600×400px
- **Описание:** Скриншот автоматической сортировки сообщения по папкам
- **Где используется:** Карточка «Автоматическая сортировка»

### feature-folders.webp
- **Размер:** ~600×400px
- **Описание:** Скриншот настройки папок и правил (sidebar/folder view)
- **Где используется:** Карточка «Гибкая настройка»

### feature-model.webp
- **Размер:** ~600×400px
- **Описание:** Скриншот настроек API (SettingsModal с выбором провайдера)
- **Где используется:** Карточка «Своя модель»

### feature-chat.webp
- **Размер:** ~600×400px
- **Описание:** Скриншот чат-интерфейса с сообщениями и вложениями
- **Где используется:** Карточка «Отправил и готово»

## Формат
- Предпочтительно WebP для оптимального размера
- Можно также PNG/JPG — в коде используется fallback
- Тёмная тема интерфейса
